## 0.9.3 - 2026/05/05

* Modernize Android build system: migrate to AGP 8.11.1 declarative plugins DSL, remove deprecated `jcenter()`, bump Gradle to 8.14
* Bump Android `compileSdk`/`targetSdk` to 36, `minSdk` to 24, Java 17 compile target
* Upgrade Kotlin to 2.2.20
* Add `namespace` to Android library build.gradle (required by AGP 7+)
* Fix Android `AndroidManifest.xml`: remove deleted v1 embedding (`io.flutter.app.FlutterApplication`), remove deleted `SplashScreenDrawable` meta-data, add `android:exported="true"` to MainActivity, retain `flutterEmbedding=2` marker
* Remove stale hand-committed `GeneratedPluginRegistrant.java` (auto-generated by Flutter tooling)
* Bump iOS minimum deployment target to 12.0
* Update Dart SDK constraint to `>=3.0.0 <4.0.0`, Flutter constraint to `>=3.16.0`
* Fix podspec placeholder metadata, sync version to 0.9.3
* Add `flutter_lints` for static analysis
* Fix deprecated `setMockMethodCallHandler` API in tests
